## 三、关键技术决策

### 3.1 知识库加载架构（⑩）

**问题**：24 个 .md 文件在 `app/masters/knowledge/`，Electron 渲染进程无法用 `fetch('file://...')` 加载本地文件。

**方案**：构建期打包（复用 `gen-prompts-builtin.py` 先例）
- `scripts/gen-knowledge-builtins.py` 读取所有 .md，Base64 编码进 `app/js/knowledge.builtins.js`
- 全局对象 `Knowledge`，暴露 `byTemp(masterKey, temperature)` / `allKeys()` / `slotsOf(key)`
- 温度三区间自动选择：
  - ≤40: `emotional` slot（温尼科特个人声音）
  - 41-70: `perspective` slot（临床视角）
  - >70: `knowledge` + `skill` slot（完整理论库）
- `package.json` 的 `predist`/`prestart` 自动执行构建脚本
- `masters.html` 显式加载（795KB 只在大师页，不进全局注入链）

### 3.2 小镜贯通架构（④）

**问题**：v3.0.0 移除了 Agent FAB，各页面小镜入口不统一。

**方案**：恢复 FAB + 利用已有全局注入链
- `preload.js` 的 `DOMContentLoaded` 已注入 `agent-tools.js` + `agent-core.js` + `agent-shell.js` + `agent.css` 到所有页面
- `agent-shell.js` 的 `buildFab()` 恢复右下角 🦐 呼吸球
- FAB CSS 已在 `agent.css` 中定义（呼吸动画 + hover + 可拖动）
- `activation.html` 排除（不需要小镜）

### 3.3 会员门控架构

**问题**：之前只有 CSS 类，没有真正的功能分层。

**方案**：四层会员系统
- `license-core.js`: `AI_TRIAL_DAYS = 60`（60 天免费试用）
- `app.js`: `featureGate(kind)` 硬门控（`ai-mindmap`/`ai-growth`/`export-full` 仅会员）
- `app.js`: `lockBadge(kind)` 视觉门控（🔒 会员 / 🔒 已过期）
- `app.js`: `membershipBadge()` 档位徽标（旗舰版/会员/试用 N 天）
- `tokens.css`: `xj-tier-badge` / `xj-lock-badge` / `xj-locked-area` 全局 CSS
- 8/8 页面接入：transcript / real-supervision / supervision / masters / report-writing / consult-notes / doc-center / billing-calendar

### 3.4 记账双栏架构（①）

**问题**：旧 `billing-shell.html` 1396 行 JS 引用 `cpanel-detail`/`cpanel-trend`/`cpanel-invoice`/`billing-aidock` 等 DOM ID，直接删元素会 JS 崩溃。

**方案**：隐藏旧容器 + 新增双栏层
- 旧三栏 `#b-rail`/`#b-center`/`#billing-aidock` 设 `display:none`，保留所有子元素
- 新增 `#dual-billing-shell` 双栏层（左收入 + 右支出）
- 新增 `initDualBillingShell()` + `renderIncomeList()` + `renderExpensePanel()` 等 15K JS
- `refreshAll()` 同步刷新双栏 + 旧视图
- Store 新增 `expenses` 数组 + 8 个 CRUD 函数

### 3.5 逐字稿 fixError Bug 修复（⑤）

**问题**：原代码先找 `err.fix`（修正词）再删掉它——逻辑反了，会把正确的词删掉。

**方案**：从出错行截取原词 → 全局替换
```javascript
// 修复前（bug）：找修正词再删掉
while ((idx = otherL.text.indexOf(err.fix, idx)) >= 0) {
  otherL.text = otherL.text.slice(0, idx) + otherL.text.slice(idx + err.fix.length);

// 修复后：从出错行截取原词，全局替换为修正词
var originalWord = l.original ? l.original.slice(err.pos, err.pos + err.len) : l.text.slice(err.pos, err.pos + err.len);
while ((idx = otherL.text.indexOf(originalWord, idx)) >= 0) {
  otherL.text = otherL.text.slice(0, idx) + err.fix + otherL.text.slice(idx + originalWord.length);
```
