## 本次内容

### 1. 知识库四项修复（commit `4cfacab`）
- **文件数护栏**：`walkUserDoc` 默认上限 500 → `KB_FILE_LIMIT=3000`，`total` 统计全部匹配文件并 `return {entries,total}`；`readUserDocMeta` 返回 `truncated`/`totalFound`/`limit`，前端常驻暖橙提示条「共 N 份，已接入前 M 份，建议拆分资料夹」——不再静默截断。
- **摘要清洗**：新增 `cleanSummary()` 去除 frontmatter / markdown 语法 / base64 / data URI / 长 URL，取首句干净散文 → 卡片底部超长乱码消失。
- **布局**：自绘标题栏去 `sticky`、面板改 `flex:1` 占满容器；三栏 `grid 190px 250px 1fr`（阅读栏主导）；阅读视图 `flex column` + 内层 `210px 1fr`，阅读栏占最大空间。
- **自检**：`scripts/self-test.js` 新增 `v3.5.1-UI-1~4` 回归（总 197 通过 / 2 既有失败无关 KB）。

### 2. H1 凭据加密 + H4 backup-config 路径过滤（commit `25aeb63`，14 文件 +467/−131）
- **H1**：`main.js` 引入 `safeStorage` 对 `LICENSE_SECRET` / `APP_PROXY_KEY` 做加解密（`encryptSecret` / `decryptSecret`），新增 `xj:encryptSecret` / `xj:decryptSecret` IPC；`license-core.js` 接入解密路径。
- **H4**：`xj:backup-config` 写入前过滤临时 / 构建产物路径，避免污染备份。
- **一致性**：`xj:readUserDocs` 改为 `fs.promises` 异步读取（与 v3.5.0-9 隐私自检异步化一致）；`extractKeywords` 改为 async（KB 关键词抽取复用）。
- 其余 v3.4.x 工作区改动一并收口（agent-core / agent-tools / ai / app / settings / sync / preload 等）。

### 3. 版本收口（commit `fbf12e0`）
