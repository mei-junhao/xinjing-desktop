## 四、架构设计

### 4.1 双模式架构

```
┌─────────────────────────────────────────┐
│           用户打开应用                   │
│            (默认入口)                    │
│              ↓                          │
│    ┌─────────────────────┐              │
│    │   对话模式           │              │
│    │   chat-home.html    │              │
│    │   → 全屏对话界面     │              │
│    │   → AgentCore.runRound()          │
│    └─────────────────────┘              │
│              ↓ 点击「专家模式」           │
│    ┌─────────────────────┐              │
│    │   专家模式           │              │
│    │   index.html + ...  │              │
│    │   → 多页面架构       │              │
│    │   → 各页面 FAB 按钮 → XinJingChat  │
│    └─────────────────────┘              │
└─────────────────────────────────────────┘
```

### 4.2 统一模块层级

```
页面 (18 个 HTML)
    ↓
XinJingChat (xinjing-chat.js)  ← 新增
    ├── AgentCore (agent-core.js)  ← 复用
    │       └── AgentTools (agent-tools.js)  ← 复用
    ├── AgentShell (agent-shell.js)  ← 复用
    └── 记忆层 (localStorage: xj_xinjing_chat_v1)

chat-home (chat-home.js)  ← 独立，但共享同一 AgentCore + 记忆 Key
```

### 4.3 记忆共享机制

- **Key**：`xj_xinjing_chat_v1`
- **范围**：所有页面共享（chat-home、XinJingChat 侧边面板、xiaojing-panel）
- **容量**：最多 50 条消息（系统消息不计入）
- **持久化**：localStorage，应用关闭后仍保留

## 五、使用说明

### 5.1 对话模式（默认入口）

打开 `chat-home.html`，直接与心镜对话：

- 输入自然语言指令，如「帮张明记一笔账」
- 点击语音按钮 🎤 进行语音输入
- 写入操作前会弹出确认卡片，预览参数后确认执行
- 工具执行后自动显示「跟进提示」，点击即可继续对话

### 5.2 专家模式

点击顶部「专家模式」按钮进入多页面界面：

- 各页面右下角有 FAB 浮动按钮（💬）
- 点击打开「心镜对话」侧边面板
- 与对话模式共享同一套对话记忆

### 5.3 撤销操作

在对话中输入「撤销」或调用 `window.undoLastWrite()`，可撤销最近一次记账写入。
