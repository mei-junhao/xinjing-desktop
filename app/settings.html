📘 如何购买 DeepSeek API
  1. 访问 platform.deepseek.com 并注册账号。
  2. 完成实名认证（需手机号，几分钟搞定）。
  3. 在「充值」页充值一点余额（如 ¥10，够用很久）。
  4. 进入「API keys」→「创建密钥」，复制生成的 sk-... 密钥（只显示一次，务必保存）。
  5. 回到心镜，点「接入高性能 AI」→ 选「已经有了」→ 选 DeepSeek → 把密钥贴给助手即可。

DeepSeek 性价比极高（deepseek-v4-flash / v4-pro 约 ¥1/百万 tokens）。其他服务商（硅基流动 / 月之暗面 / 智谱 / 通义 / 豆包 / OpenAI）流程类似。

自定义备份位置（多份容灾）

每次退出/常驻时，数据除默认备份到「文档\心镜备份」外，还会额外复制到你添加的每个位置（如移动硬盘、网盘同步目录），实现多份容灾。

说明：真正「自动发送附件」需要邮箱 SMTP 服务（nodemailer + 发件凭据），当前以唤起你的邮件客户端并预填备份路径为准。如需真正的自动发送，提供邮箱 SMTP 信息后可再加。

督导师（AI 督导）

AI 督导会读取本次会谈的逐字稿 / SOAP / 反思作为素材，用督导师身份的方法论给出督导意见。内置「温尼科特取向督导师」，你也可以添加自己的督导师身份（上传 .txt/.md 或输入提示词）。此为付费功能。

当前为免费版，督导师功能需激活后使用。激活后可在会谈页「AI 助手 → AI 督导」中使用，并在此管理身份。
