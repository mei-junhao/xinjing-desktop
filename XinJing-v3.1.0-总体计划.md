## P1 · 基础修复

### ③ 数据去重

**问题**：记账数据库存在「次结」类型与导入数据引起的会谈日期重复记录。
**方案**：
1. 编写 `scripts/dedup-sessions.js`，扫描 `Store.getSessions()`
2. 去重规则：同一 `clientId + date + sessionNumber` 存在多条时，删除 `source === 'manual'`（次结手录），保留 `source === 'import'`（导入）
3. 运行前自动备份到 `IndexedDB → kv → key='backup_pre_dedup_<timestamp>'`
4. 执行后打印去重报告（删除了 N 条，涉及 M 位来访者）
5. 在记账页面入口加「数据修复」按钮（一次性，修完隐藏）

### ⑪ 设置界面修复

**问题 A — API 配置默认走 Agent**：
- 当前 `settings.js` 有独立的 `PROVIDER_PRESETS` 与 `agent-tools.js` 的 `API_PROVIDERS` 重复定义
- 方案：删除 settings.js 中的重复定义，改为引用 `agent-tools.js` 导出的同一份预设
- 设置页「接入高性能 AI」抽屉改为：默认提示「对小镜说"接入 DeepSeek，密钥 xxx"即可配置」，保留手动配置作为高级选项

**问题 B — 版本号不正确**：
- `preload.js` 已有 `getVersion()` 读 `version.generated.js`（构建期注入）
- 但 `settings.js` 的「关于」区域可能仍读 `package.json` 的 version 或硬编码
- 方案：统一走 `window.__XJ_API__.getVersion()`，移除所有 fallback 硬编码

**问题 C — 激活状态不正确**：
- `preload.js` 有 `onLicenseState(cb)` 订阅机制
- 方案：设置页激活状态显示改为订阅式，主进程广播后实时刷新；检查 `license-core.js` 的状态判定逻辑是否正确返回 `isActivated`

## P4 · Agent 系统统一

### ④ 小镜 Agent 大幅提升

**核心问题**：小镜没有身份定义，会说超出身份的话，无法与项目数据联通。

**方案 — 四层改造**：

**层 1：身份定义（SOUL）**
- 新建 `app/js/xiaojing-identity.js`
- 身份：心镜内置助手「小镜」，是梅的临床工作伙伴，不是大师、不是督导师
- 能力边界：可以查询和操作本地数据（来访者/会话/账单/报告），可以调用 AI 辅助，但不能替代临床判断
- 严格约束：**禁止编造没有来源的数据**，涉及具体数字/来访者信息时必须先查 Store 再回答

**层 2：数据工具集（TOOLS）**
- 小镜获得只读数据工具（复用 agent-tools.js 现有工具）：
  - `query_clients` — 查来访者列表
  - `query_sessions` — 查会话记录
  - `query_billing` — 查账单/欠费
  - `query_reports` — 查报告
  - `query_supervisions` — 查督导记录
- 写工具（需确认）：
  - `create_session` — 记一笔
  - `mark_paid` — 标记已收
  - `navigate_to` — 跳转页面

**层 3：页面上下文（CONTEXT）**
- 每个页面初始化时注入 `window.__XJ_CONTEXT__`
- 小镜系统提示动态拼接：基础身份 + 当前页面上下文 + 可用工具列表
- 页面上下文示例（记账页）："当前在记账页面，用户可以问你查欠费、记账、月结。如果用户说'帮林小满记7月10号会谈300块次结没付'，你应该调用 create_session 工具。"

**层 4：反幻觉约束（GUARD）**
- 系统提示硬约束：
  ```
  【数据约束 — 最高优先级】
  - 涉及来访者姓名、费用金额、会话次数、日期等具体信息时，必须先调用查询工具获取真实数据，不得凭记忆回答。
  - 如果查询结果为空或不足，明确说"我查了一下，没有找到相关记录"。
  - 绝对禁止编造来访者姓名或数字。
  - 回答"谁欠费最多"类问题时，必须调用 query_billing 工具排序后回答。
  ```

**各页面小镜功能对照**：

| 页面 | 上下文 | 小镜能力 |
|------|--------|----------|
| 首页 | 今日咨询数/待办/欠费 | 每日问候 + 对话修改日程 + 快捷导航 |
| 咨询记录 | 当前来访者+模式 | 辅助展开 APA/SOAP/DAP 字段 + 临床材料问答 |
| 撰写报告 | 当前步骤+来访者 | AI 填本步/全部 + 模板分析 + 逐段填充 |
| AI 督导 | 当前来访者+取向 | 快捷按钮（整体印象/深化/润色/技术建议/移情/邀请大师）+ 自由对话 |
| 记账 | 月度概览+欠费 | 自然语言记账 + 查询欠费 + 月结提醒 |
| 逐字稿整理 | 当前来访者+原文 | 错误标注引导 + 批量替换 + 记忆建议 |
