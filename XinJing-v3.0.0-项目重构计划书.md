**日期**：2026-07-13
**设计系统**：静谧留白 calm（保留）+ 去侧栏化 + 小镜 Agent 贯通全部页面
**Design Read**：产品级咨询师管理工作台 × 单人执业 × 温尼科特取向心理动力学 × editorial-calm × 暖灰单色 + 紫调 accent × Noto Serif SC + system sans

## 一、架构级变更

### 1.1 去侧边栏
- **现状**：240px 左侧固定导航栏占用首屏 20% 宽度，视觉拥挤
- **v3.0**：侧边栏完全移除。首页作为"模块中枢"，以 4×2 卡片网格陈列所有功能入口。每个子页面独立，通过首页进入
- 返回键：每个子页面顶部保留简洁返回箭头 → 回到首页

### 1.2 小镜 Agent 贯通
- **首页**：小镜是常驻智能面板（非浮窗），位于页面右侧 380px 固定栏
  - 每日个性化欢迎语（基于日历/咨询数据动态生成）
  - 今日咨询汇总提醒："今天有 3 节咨询，已完成 2 节，王芳取消了 16:00 的咨询对吗？"
  - 可对话式修改日程
- **子页面**：每个页面右下角固定"小镜"图标，点击展开该页面专属 AI 坞（联系上下文，非通用 Agent）
  - 咨询记录页 → 辅助整理逐字稿
  - 写报告页 → 辅助搜索跨节信息
  - 督导页 → 辅助生成印象/深化
  - 记账页 → 自然语言记账

### 1.3 页面结构重定义

| 旧页面 | v3.0 页面 | 说明 |
|--------|----------|------|
| index.html（侧栏+工作台） | index.html（模块中枢+小镜） | 无侧栏，卡片式入口 + 今日概览 |
| consultations.html（三栏） | consult-notes.html（快速咨询记录） | APA 结构化提示 + 小镜问答式录入 |
| session.html + client-detail.html | report-writing.html（案例报告） | 左来访者列表 + 中全历史 + 右 AI 辅助 |
| supervision.html | supervision.html（督导） | 复用方案1结构，方案5为会员功能 |
| billing-shell.html | billing.html（账单） | 重构 + 清除数据功能 |
| masters.html | masters.html（大师对话） | 保留现有重写版本 |

### 2.2 咨询记录（快速整理）
- **定位**：做完咨询后的轻量记录入口，非完整编辑
- **APA 结构化提示**：引导用户按 5 个维度简要梳理（主诉/行为观察/情绪状态/关键对话片段/下次方向）
- **小镜问答式录入**：复用督导方案 2 的双栏对照 —— 左栏书写区，右栏 AI 对话面板
- **不关联完整 SOAP**——那是 report-writing 页面的职责

### 2.3 撰写报告（呈报督导师用）
- **定位**：面向督导师的完整案例报告，可导出 Word
- **结构**：三栏 —— 左 280px 来访者列表（含全部历史），中栏全历史时间线（可快捷增节），右栏 Word 式编辑器
- **关键功能**：
  - 选来访者 → 显示该来访者全部节次+逐字稿+SOAP+DAP+反思
  - 可在此快捷增加咨询节数
  - AI 辅助：录音转文字、逐字稿整理
  - 上传案例报告格式模板 → 界面变为引导式填写
  - 会员：一键 AI 生成完整报告
  - AI 坞可跨节搜索匹配信息："帮我找林小满第 1-5 节中关于母亲的所有描述"
  - 点"整理报告"→ 勾选基于哪些节次 → AI 整合 → 生成本地 Word 文档

### 2.4 督导
- **基础版**：方案 1（分阶段工作台）—— 选来访者→提交材料→AI 印象→深化
- **会员版**：方案 5（多大师画布）—— AI+多位大师+真人督导卡片式并行
- 同上，坞 sticky 固定，不随文本滚出视野

### 2.5 账单
- 保留现有三栏记账结构
- 新增 **「清除数据」** 按钮：点击弹出确认对话框，需手动输入"确认清除"四个字才能执行
- 清除范围：**仅来访者账单信息**（sessions 中 billing 相关 + clients 中 billing 相关）
- **不清除**：督导记录、大师对话、设置

## 六、设计规范（基于 taste-skill + minimalist-skill）

- 调色板：暖灰单色 `--bg: #F6F8FC` + accent 紫 `#8B93C7`（保留现有 calm 令牌）
- 字体：Noto Serif SC（标题）+ system sans（正文）
- 卡片：`border: 1px solid var(--border)` + `border-radius: 14px` + 无重阴影
- 按钮：accent 色 solid + 8px radius
- 间距：宏间距 32-40px 段间，微间距 12-16px 项间
- 无渐变、无玻璃态、无 emoji（小镜会话区除外）
- 页面最大宽度 1200px 居中
